# Telecom-customer-churn-analysis-using-Power-BI-
# Customer Churn Analysis — Power BI Dashboard

## Overview
Interactive Power BI dashboard analyzing telecom customer churn 
across 7,043 customers using the IBM Telco dataset.

## Key Insights
-## Key Insights
- 26.5% Churn Rate — Over 1 in 4 customers churned,
  representing a significant revenue risk.
- Month-to-month contracts drive churn — 55% of churned
  customers were on monthly contracts vs 24% on two-year plans,
  suggesting long-term contracts improve retention.
- Fiber Optic users churn most — 44% of churned customers
  used Fiber Optic service, indicating potential quality or
  pricing dissatisfaction.
- Electronic payment correlates with churn — Highest churn
  rate observed among electronic check users.
- Average Monthly Charge: $64.76 — Churned customers paid
  higher monthly fees on average, linking price sensitivity
  to churn behavior.
- Gender has minimal impact — Male (49.52%) vs Female
  (50.48%) churn is nearly equal, ruling out gender as
  a churn driver.

## Tools Used
- Power BI Desktop
- DAX Measures
- Power Query (ETL & Data Cleaning)

## Dataset
IBM Telco Customer Churn Dataset — 7,043 records, 33 variables
Source: Kaggle
